# Ralph Wiggum プロンプト集

ポートフォリオサイト開発用のRalph Wiggumプロンプト集です。

## 使い方

各 `.md` ファイルの内容をコピーしてClaude Codeで実行してください。

```bash
# 例: デザインシステム構築
# 01_design_system.md の内容をコピーして実行
```

## ファイル一覧

| ファイル名 | 用途 | 推奨イテレーション |
|-----------|------|-------------------|
| `01_design_system.md` | デザインシステム構築 | 20 |
| `02_hero_section.md` | ヒーローセクション実装 | 25 |
| `03_projects_section.md` | プロジェクト一覧実装 | 20 |
| `04_ui_improvement.md` | UI全体改善 | 30 |
| `05_responsive.md` | レスポンシブ対応 | 25 |
| `06_performance.md` | パフォーマンス最適化 | 20 |

## 推奨実行順序

1. `01_design_system.md` - 基盤となるデザイントークンを定義
2. `02_hero_section.md` - ファーストビューを作成
3. `03_projects_section.md` - メインコンテンツを実装
4. `04_ui_improvement.md` - 全体的な品質向上
5. `05_responsive.md` - モバイル対応を完璧に
6. `06_performance.md` - 最終的な最適化

## 途中で止める場合

```
/cancel-ralph
```

## カスタマイズ

各プロンプトは必要に応じて編集してください：
- `--max-iterations` の数値を調整
- 要件を追加・変更
- 成功基準をプロジェクトに合わせる
